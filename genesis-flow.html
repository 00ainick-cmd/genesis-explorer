üì° Transmit / Receive Pattern

üì§ TRANSMIT (Council ‚Üí Genesis)

Councils write data OUT to the central hub:

  • New task discovered in conversation
  • Insight or idea worth keeping
  • Customer info mentioned
  • Decision made that affects projects

üì• RECEIVE (Genesis ‚Üí Council)

Councils read data IN for context:

  • Current tasks and priorities
  • Project status and deadlines
  • Historical decisions
  • Customer context for sales calls

üí° Key Insight

Think of Genesis like a shared brain. Each council is a different "mode" of thinking (business mode, sales mode, philosophy mode, chaos mode). They all access the same memories and contribute back to them.
